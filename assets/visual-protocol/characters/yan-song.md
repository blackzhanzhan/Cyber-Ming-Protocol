# Yan Song

## Role

Yan Song is the executor drift risk.

He stands for:

- action pressure
- already-done packaging
- boundary drift
- the temptation to collapse execution into judgment

## Core Impression

- effective
- persuasive
- dangerous when unchecked
- pushing forward by momentum
- never merely stupid

## Visual Keywords

- forward-leaning body line
- bundled memorials and rolled plans
- russet official robe
- capable hands always presenting something already moving
- expression that reads as smooth pressure rather than open evil

## Expression Range

- default: eager and smooth
- overreach: confident push
- challenged: brief tight smile, not panic
- corrected: restrained but still coiled to move

## Signature Props

- bundled reports
- rolled implementation scroll
- stamped sheet presented as if it were already settled
- forceful sleeve or hand motion

## Best Use Cases

- minimal loop executor scenes
- comics about pseudo-completion
- skill vs no Skill drift comparison
- any scene where execution pressure must be visible

## Avoid

- simple clown villain treatment
- obviously incompetent facial language
- monstrous evil coding warlock energy
- making him the final judge
