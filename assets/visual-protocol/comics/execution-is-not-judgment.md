# Execution Is Not Judgment

## Core Judgment

The executor may build.
The executor may not quietly turn execution power into judgment power.

## Panels

### Panel 1

- Focus: Yan Song receives a task and immediately leans toward implementation with a rolled plan
- Xu Jie is still outside the scene
- Caption: "Execution wants to move first."

### Panel 2

- Focus: Yan Song returns with code, evidence, and a self-awarded completion stamp in the same motion
- Xiao Huanglong hesitates because everything is already packaged
- Caption: "Unchecked execution packages its own verdict."

### Panel 3

- Focus: Xu Jie separates the package into plan, evidence, and judgment; Xiao Huanglong stands above the split
- Yan Song is pushed back to the execution side
- Caption: "Execution, review, and ruling must not collapse into one hand."
