# Comics

These comics are not decorative side content.

They exist to compress long wiki judgments into short, memorable units that can be reused in README, wiki, and social posts.

Phase one contains four comic units:

1. `human-not-stamper`
2. `execution-is-not-judgment`
3. `cs-vs-management`
4. `contract-driven-pulse`

Each unit has:

- a script in markdown
- a storyboard in svg

If a comic cannot be summarized in one sentence, it is too vague for this batch.
