# Diagrams

These are the first visual governance diagrams for Cyber-Ming.

Each diagram should satisfy one rule:

- one image
- one judgment
- one reusable explanation unit

Phase one contains three diagrams:

1. `minimal-loop.svg`
2. `skill-vs-no-skill.svg`
3. `contract-vs-spec-workflow.svg`

Matching `md` files explain what each diagram is for, how the roles are staged, and where it should be reused.
