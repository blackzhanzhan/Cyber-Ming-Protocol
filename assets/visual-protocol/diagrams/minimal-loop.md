# Minimal Loop Diagram Brief

## Core Judgment

The minimal loop is not two agents collaborating freely.
It is a human-routed loop in which execution and audit stay separate.

- wiki Layer 1 anchor image
- repost image for explaining human-centered routing

- Xu Jie: auditor
