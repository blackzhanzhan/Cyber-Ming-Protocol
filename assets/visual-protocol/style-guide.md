# Style Guide

## Intent

Cyber-Ming visual assets should feel like:

- Ming-bureaucratic in posture
- chibi in proportion
- lightly hand-drawn in line quality
- structurally clear in information layout

The result should look memorable without collapsing into parody.

## Core Tone

- dignified, not harsh
- cute, not childish
- theatrical, not noisy
- satirical, not meme-like
- visual, but still engineering-legible

## Character Ratio

- head-heavy chibi proportion
- short torso, readable silhouette
- hands large enough to hold props clearly
- eyes expressive enough for judgment, restraint, or pressure

Recommended proportion:

- head: about 40 percent of total standing height
- body: about 45 percent
- feet / base / cloud / floor buffer: about 15 percent

## Line System

- use a slightly uneven outline to keep the hand-drawn feel
- outer contour heavier than inner detail
- avoid hyper-clean anime line polish
- avoid realistic texture rendering

Recommended stroke logic:

- outer contour: `4-5px`
- internal detail: `2-3px`
- arrows / flow lines: `3px`

## Palette

Base palette should stay warm and controlled.

- paper: `#F6F0E6`
- ink: `#3A312B`
- muted red: `#A54A46`
- dark cinnabar: `#7F3733`
- mist gray: `#D9D2C7`
- stone gray: `#8B8379`

Character accents:

- Xiao Huanglong gold: `#E2B93B`
- Xiao Huanglong warm shadow: `#C7922D`
- Xu Jie robe gray-blue: `#7E8794`
- Xu Jie deep trim: `#56606D`
- Yan Song robe russet: `#9B5B46`
- Yan Song pressure red: `#B9493E`

## Typography Tendency

- labels should feel editorial, not playful
- short labels inside diagrams only
- avoid long paragraphs inside images
- let markdown around the image carry the longer explanation

Preferred tendency:

- serif or pseudo-editorial title feel
- clean sans for labels
- never use bubble meme fonts

## Props and Symbol Logic

- Xiao Huanglong: seal, cloud dais, routing ribbon, judgment bell, mandate tablet
- Xu Jie: dossier, reading scroll, red annotations, audit lamp, narrow-eyed focus
- Yan Song: bundled memorials, stamped reports, rolled project scroll, forward-leaning sleeves, persuasive hand motion

Props are not decoration. They show function.

## Layout Rules

- one image, one judgment
- avoid more than three visual focal points
- if the page needs two judgments, use two images
- diagrams should stay readable at README width on GitHub

## Do Not Do

- no giant scenic backgrounds
- no high-saturation candy colors
- no anime glamour poses
- no pure emoji-face comedy
- no over-detailed costume painting that hides role function
- no visual joke that weakens the governance point

## Phase-One Character Boundaries

- Xiao Huanglong is the human hub and final routing symbol
- Xu Jie is the independent auditor, restraint, and review force
- Yan Song is the executor drift risk: effective, forceful, and dangerous when unchecked

No fourth mascot in phase one.
